REWE Article Crawler
=====================

Requirements
-------------
**Python 2.7**
```
sudo apt-get install python
```

Packages: psycopg2 (postgres driver), BeautifulSoup 4 and lxml (HTML parsers)
```
$ pip install psycopg2
$ pip install lxml
$ pip install beautifulsoup
```

Run
------
```
$ python rewe.py
```

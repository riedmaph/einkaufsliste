Special Offer Crawlers
======================

 - **edeka.py**: offers from edeka.de (all markets)
 - **rewe-api.py**: offers from rewe-api.de (all markets)

Note: depends on markets (need to be crawled first).

Requirements
-------------
**Python 2.7**
```
sudo apt-get install python
```

Run
------
```
$ python <script.py>
```
e.g.
```
$ python rewe.py
```

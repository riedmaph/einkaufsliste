Market Crawlers
======================

 - **edeka.py**: markets from edeka.de
 - **rewe-api.py**: markets from rewe-api.de

Note: execution deletes all corresponding offers!

Requirements
-------------
**Python 2.7**
```
sudo apt-get install python
```

Run
------
```
$ python <script.py>
```
e.g.
```
$ python rewe.py
```
